### ExpressionDataPath / 計算式データパス 列挙型

計算式の入力とするデータのパスを表す。XMLにおけるXPathのような概念。

```plantuml
hide empty members
skinparam defaultFontName Monospaced

enum ExpressionDataPath {
    + コンテナジョブセット/コンテナジョブ
    + ROROジョブセット/ROROジョブ
    + 車輌/放射能検査履歴
    + 車輌/ヤードサマリ ※仮
    + 車輌/ヤード搬入履歴
    + コンテナジョブセット/コンテナジョブ
    + コンテナジョブセット/コンテナジョブ/車輌
    + コンテナジョブセット/コンテナジョブ/車輌/放射能検査履歴
    + コンテナジョブセット/コンテナジョブ/車輌/ヤードサマリ ※仮
    + コンテナジョブセット/コンテナジョブ/車輌/ヤード搬入履歴
    + ROROジョブセット/ROROジョブ
    + ROROジョブセット/ROROジョブ/車輌/放射能検査履歴
    + ROROジョブセット/ROROジョブ/車輌/ヤードサマリ ※仮
    + ROROジョブセット/ROROジョブ/車輌/ヤード搬入履歴
    + 車輌
    + 車輌/ヤード
    + 車輌/ヤード搬入履歴/ヤード
    + コンテナジョブ/ヤード
    + ROROジョブ/ヤード
    + etc... TBC.
}
```
